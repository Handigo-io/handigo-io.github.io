ABOUT HANDIGO

Handigo is a gesture-based computer control system developed as a thesis project. The application enables users to interact with desktop and laptop computers hands-free, serving as an alternative to traditional input devices such as a mouse and keyboard.

Using a standard webcam, Handigo detects and interprets hand gestures in real time and converts them into corresponding system commands. The system is designed to promote touchless interaction, improved accessibility, and enhanced user comfort.

Guide For Installation

Handigo is designed exclusively for desktop and laptop computers. The application supports Windows systems using a standard webcam. Mobile devices and gaming consoles are not supported.

The system uses computer vision to detect predefined gestures and convert them into mouse and keyboard commands in real time.

This project is an academic thesis system. Reverse engineering, modifying, or redistributing the source code without permission is not permitted.
